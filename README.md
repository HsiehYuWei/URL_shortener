# 縮網址

## 介紹
提供縮網址功能

## 使用手冊

1. 請先確認有安裝 node.js 與 npm

2. 將專案 clone 到本地

3. 在本地開啟之後，透過終端機進入資料夾，輸入：
```
npm install
```

4. 安裝完畢後，繼續輸入：
```
npm run dev
```
5. 若顯示此行訊息則代表順利運行
```
Listening on http://localhost:3000
```
6. 欲結束使用
```
ctrl + c
```
## 開發工具
* Node.js 10.15.0
* Express 4.18.2
* Express-Handlebars 4.2.0
